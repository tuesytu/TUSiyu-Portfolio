# NOGGIN Playful Systematic 4.1

## 核心视觉理念

- 手绘与系统的碰撞：粗犷标题字形 + 规则网格背景 + 强分段线
- 触感审美：像实验记录本、课程档案、作品调研页
- 结构化布局：按简历阅读逻辑分为个人介绍、技能、教育、项目、实习、联系

## 颜色系统

- 主背景：`#F2E7DF`
- 模块绿：`#96A388`
- 模块橙：`#E8A08A`
- 模块黄：`#F9D371`
- 强调橙：`#D06444`
- 标签黄：`#FFC12D`
- 墨色：`#222222`
- 背景网格线：`rgba(34, 34, 34, 0.08)`

## 字体系统

### 标题字

- 用途：姓名、章节标题、学校名、项目名、页脚口号
- 基础：`Arial Black`
- 风格：全大写、粗糙边缘、强压缩感
- 处理：统一叠加 `#rough-edge` SVG filter
- 建议规格：主标题 `5.5rem`，章节标题 `4rem`，内容标题 `2.4rem`

### 系统字

- 用途：导航、年份、标签、辅助说明
- 基础：`Space Mono`
- 规格：`0.75rem` 到 `0.9rem`
- 风格：高对比、偏档案化、适合中英混排

### 中文系统字

- 用途：正文与按钮中的中文
- 基础：`PingFang SC`
- 规则：与 `Space Mono` 混用，不做额外装饰

## 布局规范

- 顶部导航高度：`72px`
- 全站基础留白：`8rem 4rem`
- 内容容器宽度：`1200px`
- 背景网格：`60px * 60px`
- 每个大区块必须有明确底边线
- `About / Education / Projects / Internships` 采用整屏大段落切换

## 组件规范

### 导航栏

- 左上角固定为：`Portfolio`
- 中间导航采用中英并置写法
- 当前文案：
  - `关于我 ABOUT ME`
  - `作品集 WORKS`
  - `其他 OTHER`
- 右上角语言切换器使用不规则手绘边框按钮

### 技能圆

- 白底黑边圆形
- Hover 时当前圆放大，其他圆被推开并降低透明度
- 外圈动态弹出工具标签

### 教育标签组

- 黄色主标签 + 橙色隐藏说明标签
- Hover 时主标签左移，细节标签出现

### 经历列表

- 左列：时间 + 职能
- 右列：标题 + 要点
- 左右分栏之间必须有竖线
- 要点前使用橙色圆点

## 当前页面结构

- Hero：姓名、联系方式、滚动提示
- About：技能概括
- Education：两段教育经历
- Projects：项目经历
- Internships：实习经历
- Footer：联系收尾

## 当前实现要求

- 继续优先使用静态 HTML 快速预览
- 页面视觉以你本次提供的 HTML 为准
- 后续新增真实作品详情时，也要沿用这套分段、字体、边框、网格和配色逻辑
