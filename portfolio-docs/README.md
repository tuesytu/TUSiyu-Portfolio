# Portfolio Docs

这是当前作品集网站的**唯一资料源目录**。

以后关于这个网站的需求、内容、视觉规范、双语文案、以及你在对话中持续确认的偏好，统一放在这里。

## 文件说明

- `PRD.md`
  - 产品需求文档
  - 说明网站要做什么、有哪些模块、有哪些功能

- `CONTENT.md`
  - 作品集内容素材
  - 包含简历信息、项目名称、项目简介、其他作品名称等

- `TRANSLATIONS.md`
  - 中英文文案映射
  - 以后所有双语修改优先改这里

- `STYLE-SYSTEM.md`
  - 当前视觉设计系统
  - 现在使用的是 `NOGGIN Playful Systematic / 4.0`

- `PROMPT-MEMORY.md`
  - 长期记忆型 Markdown
  - 用来记录你在后续对话中不断补充的偏好、方向、禁忌、改动结论
  - 这是最接近“像 skill 一样持续记住”的文件

- `OPEN-QUESTIONS.md`
  - 尚未补齐的信息
  - 后续需要你继续提供的内容

## 使用规则

1. 后续做这个作品集网站时，优先读取本目录文件。
2. 如果需求发生变化，先更新这里，再改网页。
3. `PROMPT-MEMORY.md` 作为持续迭代记忆，后面每轮重要结论都应写回这里。

## 当前主入口

如果你只想看最重要的两个文件，先看：

1. `PRD.md`
2. `PROMPT-MEMORY.md`
